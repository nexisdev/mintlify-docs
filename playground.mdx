# Nex-T1 API Playground

<Info>
This playground is generated from the [OpenAPI specification](https://api.nex-t1.ai/docs) and provides interactive testing for all Nex-T1 endpoints.
</Info>

## Base URL Options

You can test against two environments:

| Environment | Base URL | Description |
|------------|----------|-------------|
| **Production** | `https://api.nex-t1.ai` | Live production API with real data |
| **Local Development** | `http://localhost:8000` | Your local instance for testing |

<Tip>
Use the **Base URL switcher** at the top of each API request to toggle between production and local development.
</Tip>

## Authentication

Most endpoints require JWT bearer authentication:

1. **Get a token**: Call the `/api/v1/auth/register` or `/api/v1/auth/login` endpoint
2. **Set authentication**: Click "Set Authentication" in the playground
3. **Add token**: Paste `Bearer <your_token>` in the authorization field

<Warning>
Keep your JWT tokens secure and never share them publicly. Tokens expire after a set period and must be refreshed.
</Warning>
