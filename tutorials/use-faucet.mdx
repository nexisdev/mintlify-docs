This tutorial walks you through obtaining testnet tokens needed for development on Nexis Appchain. You'll learn how to get NZT tokens, bridge ETH from Base Sepolia, add the network to MetaMask, and verify your balances.

### Step 3: Bridge Base Sepolia ETH to Nexis

**Using the Nexis Bridge:**

```bash
# Option 1: Web Interface
Visit: https://bridge.nex-t1.ai

# Option 2: CLI Tool
npx @nexis/bridge-cli deposit \
  --from base-sepolia \
  --to nexis-testnet \
  --amount 0.1 \
  --token ETH
```

**Manual Bridge via Smart Contract:**

```javascript
const { ethers } = require('ethers');

// Connect to Base Sepolia
const provider = new ethers.JsonRpcProvider('https://sepolia.base.org');
const wallet = new ethers.Wallet(PRIVATE_KEY, provider);

// Nexis L2 Bridge Contract on Base Sepolia
const bridgeAddress = '0xBridgeContractAddress';
const bridgeABI = [
  'function depositETH(uint32 _minGasLimit) external payable'
];

const bridge = new ethers.Contract(bridgeAddress, bridgeABI, wallet);

// Bridge 0.1 ETH to Nexis
const tx = await bridge.depositETH(200000, {
  value: ethers.parseEther('0.1')
});

console.log('Bridge transaction:', tx.hash);
await tx.wait();
console.log('‚úÖ Bridged successfully!');
```

<Note>
**Bridge Time:** Deposits from Base Sepolia to Nexis typically take 2-5 minutes.
</Note>

## Part 4: Automated Faucet Script

Create a script to automatically request tokens:

```javascript auto-faucet.js
const { ethers } = require('ethers');
const axios = require('axios');

async function requestNexisFaucet(address, amount = '10') {
  console.log('üö∞ Requesting', amount, 'NZT from faucet...');

  try {
    const response = await axios.post('https://faucet.nex-t1.ai/api/claim', {
      address: address,
      amount: amount
    });

    if (response.data.success) {
      console.log('‚úÖ Success! Transaction:', response.data.txHash);
      console.log('   Amount:', ethers.formatEther(response.data.amount), 'NZT');

      // Wait for confirmation
      const provider = new ethers.JsonRpcProvider('https://testnet-rpc.nex-t1.ai');
      console.log('   Waiting for confirmation...');

      await provider.waitForTransaction(response.data.txHash);
      console.log('   ‚úÖ Confirmed!');

      // Check balance
      const balance = await provider.getBalance(address);
      console.log('   New balance:', ethers.formatEther(balance), 'NZT');
    } else {
      console.error('‚ùå Failed:', response.data.message);
    }
  } catch (error) {
    if (error.response) {
      console.error('‚ùå Error:', error.response.data.message);
      if (error.response.data.retryAfter) {
        const hours = error.response.data.retryAfter / 3600;
        console.log(`   Try again in ${hours.toFixed(1)} hours`);
      }
    } else {
      console.error('‚ùå Request failed:', error.message);
    }
  }
}

// Usage
const address = process.argv[2];
const amount = process.argv[3] || '10';

if (!address) {
  console.error('Usage: node auto-faucet.js <address> [amount]');
  process.exit(1);
}

requestNexisFaucet(address, amount);
```

**Run the script:**

```bash
# Install dependencies
npm install ethers axios

# Request 10 NZT
node auto-faucet.js 0xYourAddress

# Request 50 NZT
node auto-faucet.js 0xYourAddress 50
```

## Part 5: Balance Monitoring

Create a balance checker script:

```javascript check-balances.js
const { ethers } = require('ethers');

async function checkBalances(address) {
  console.log('üí∞ Balance Check for:', address);
  console.log('='.repeat(60));

  // Nexis Testnet
  console.log('\nüîµ Nexis Appchain Testnet:');
  const nexisProvider = new ethers.JsonRpcProvider('https://testnet-rpc.nex-t1.ai');
  const nexisBalance = await nexisProvider.getBalance(address);
  console.log('   NZT Balance:', ethers.formatEther(nexisBalance));

  const nexisTxCount = await nexisProvider.getTransactionCount(address);
  console.log('   Transactions:', nexisTxCount);

  // Base Sepolia
  console.log('\nüü† Base Sepolia:');
  const baseProvider = new ethers.JsonRpcProvider('https://sepolia.base.org');
  const baseBalance = await baseProvider.getBalance(address);
  console.log('   ETH Balance:', ethers.formatEther(baseBalance));

  const baseTxCount = await baseProvider.getTransactionCount(address);
  console.log('   Transactions:', baseTxCount);

  // Ethereum Sepolia
  console.log('\nüü£ Ethereum Sepolia:');
  const sepoliaProvider = new ethers.JsonRpcProvider('https://rpc.sepolia.org');
  const sepoliaBalance = await sepoliaProvider.getBalance(address);
  console.log('   ETH Balance:', ethers.formatEther(sepoliaBalance));

  const sepoliaTxCount = await sepoliaProvider.getTransactionCount(address);
  console.log('   Transactions:', sepoliaTxCount);

  console.log('\n' + '='.repeat(60));

  // Check if more tokens needed
  const minRequired = ethers.parseEther('1');
  if (nexisBalance < minRequired) {
    console.log('\n‚ö†Ô∏è  Low NZT balance! Visit the faucet:');
    console.log('   https://faucet.nex-t1.ai');
  }
}

const address = process.argv[2];
if (!address) {
  console.error('Usage: node check-balances.js <address>');
  process.exit(1);
}

checkBalances(address);
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Faucet says 'Rate limit exceeded'" icon="clock">
**Problem:** You've reached the daily limit.

**Solutions:**
- Wait for cooldown period (check response message)
- Use Discord bot for higher limits
- Ask in Discord #faucet-requests for manual drip
- Use a different address if testing
  </Accordion>

  <Accordion title="Tokens not appearing in wallet" icon="eye-slash">
**Problem:** Balance not showing in MetaMask.

**Checks:**
1. Are you on Nexis Testnet? (Check network dropdown)
2. Is the transaction confirmed? (Check explorer)
3. Has enough time passed? (Wait 30 seconds)
4. Try refreshing MetaMask (lock/unlock)

**Verify on explorer:**
```
https://testnet.nex-t1.ai/address/YOUR_ADDRESS
```
  </Accordion>

  <Accordion title="Cannot add Nexis network" icon="network-wired">
**Problem:** MetaMask won't add the network.

**Solutions:**
- Update MetaMask to latest version
- Clear browser cache
- Try manual addition instead of automatic
- Check RPC URL is exactly: `https://testnet-rpc.nex-t1.ai`
- Verify Chain ID is exactly: `84532`
  </Accordion>

  <Accordion title="Bridge stuck pending" icon="bridge">
**Problem:** Bridge transaction not completing.

**Investigation:**
```bash
# Check transaction on Base Sepolia
https://sepolia.basescan.org/tx/YOUR_TX_HASH

# Check if it arrived on Nexis
https://testnet.nex-t1.ai/address/YOUR_ADDRESS
```

**Common causes:**
- Insufficient gas on source chain
- Network congestion (wait 10-15 minutes)
- Wrong destination address
- Bridge contract paused (check Discord)
  </Accordion>

  <Accordion title="RPC not responding" icon="server">
**Problem:** Cannot connect to Nexis RPC.

**Test connectivity:**
```bash
# Test main RPC
curl -i https://testnet-rpc.nex-t1.ai

# Try backup RPC
curl -i https://testnet-rpc.nex-t1.ai
```

**Solutions:**
- Check your internet connection
- Try backup RPC URL
- Wait if network is upgrading (announced in Discord)
- Use VPN if region-blocked
  </Accordion>
</AccordionGroup>
